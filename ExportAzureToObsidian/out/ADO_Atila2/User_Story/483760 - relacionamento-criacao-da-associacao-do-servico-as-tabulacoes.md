## 9. Descrição (Abaixo vem do Azure DevOps)

Eu, como Gestor aluno,  


Quero associar ou criar tabulações para cada Serviço,  
Para garantir que, durante o atendimento, os atendentes possam categorizar corretamente o motivo da solicitação com base no tipo de serviço prestado, permitindo a padronização das análises e relatórios.

RN01: Durante a criação ou edição de um Serviço, o sistema deve permitir a associação de uma ou mais Tabulações (Categoria/Subcategoria) específicas para aquele Serviço. 

RN02: As tabulações devem conter no maximo três níveis hierarquicos de classificação.

RN03: É obrigatória a associação de pelo menos uma Tabulação para que o Serviço possa ser ativado/publicado. 

RN04: Tabulações não associadas ao Serviço devem ficar indisponíveis para seleção no momento do encerramento do atendimento.
